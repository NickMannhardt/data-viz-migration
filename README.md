## Getting Started
After pulling from main, run
```
cd frontend
npm install
```
and
```
cd server/src
pip install -r requirements.txt
```

Then to start the development server for the backend run
```
cd server/src
python app.py
```

to start the development server for the frontend run
```
cd frontend
npm run dev --open
```
